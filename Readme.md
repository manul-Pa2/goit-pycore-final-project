# Amadeus_bot — це консольний бот для керування контактами.

Підтримує збереження даних, роботу з телефонами, email, адресами, датами народження та пошук найближчих днів народження.

# Amadeus_bot створений командою №6 для скаладання екзамінаційного проекту в рамках курсу Python.

## Інструкція по запуску з термінала

1. Переконайтесь, що на вашому комп'ютері встановлено "Python 3.10", або новіше...

2. Перейдіть у директорію проєкту:

* cd project

3. Запустіть бота командою:

* python main.py

4. При першому запуску автоматично створиться файл "addressbook.pkl" з порожньою адресною книгою.


## Можливості

Бот надає такі функції:

Додавання контактів і окремих полів:

* номера телефону
* email
* адреси
* дати народження
* Зміна існуючих даних

Перегляд:

* номера телефону
* email
* адреси
* дати народження
* повної інформації про контакт

- Виведення всіх контактів
- Пошук найближчих днів народження
- Довідка щодо доступних команд (help)
- Автоматичне збереження даних при завершенні роботи

## Структура проєкту

project/
│
├── main.py               # Точка входу
├── addressbook.pkl       # Файл збереження (створюється автоматично)
└── src/
├── handlers.py       # Реалізація всіх команд-обробників
├── entities.py       # AddressBook і сутності контактів
└── ...               # Додаткові модулі логіки

## Збереження даних

* Проєкт використовує pickle для збереження екземпляра AddressBook:

* при запуску дані читаються з addressbook.pkl;

* при виході (close або exit) дані автоматично зберігаються.

* Якщо параметр filename дорівнює None або "", програма працює без збереження.

## Запуск

python main.py

* При першому запуску буде створено порожню адресну книгу.

## Команди

* Нижче подано команди, які підтримуються в handlers:

| Команда                      | Опис                                |
| ---------------------------- | ----------------------------------- |
| hello                        | Привітання                          |
| add ...                      | Додати контакт або номер телефону   |
| change ...                   | Змінити контакт або номер телефону  |
| add-email / change-email     | Додати або змінити email            |
| add-birthday                 | Додати дату народження              |
| add-address / change-address | Додати або змінити адресу           |
| phone / show-phone           | Показати номер телефону             |
| show-email                   | Показати email                      |
| show-address                 | Показати адресу                     |
| show-contact                 | Показати всю інформацію про контакт |
| all                          | Показати всі контакти               |
| birthdays                    | Вивести найближчі дні народження    |
| ? або help                   | Довідка                             |
| close / exit                 | Вихід зі збереженням                |

Фактичну логіку команд реалізовано у файлі src/handlers.py.

## Як це працює

* Основний цикл застосунку:

Виводить довідку.
Очікує вводу користувача.
Розбирає команду через parse_input.
Знаходить відповідний обробник у handlers.
Виконує команду та виводить результат.
Під час завершення роботи — зберігає дані.

Усі дані зберігаються всередині об’єкта AddressBook.
